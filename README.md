Nova AI is a Python-based hybrid AI assistant that combines local AI models, cloud AI, voice interaction, web search, and a Flask web UI.

This repository is structured so that anyone can clone it and run the project on their own PC.

## Features
- Voice & Text interaction
- Local AI using Ollama
- Cloud AI (Gemini + OpenRouter)
- Model switching
- Web search (RAG)
- YouTube music playback
- Flask Web UI
- Rich terminal UI

## Installation
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Install Ollama
5. Run main.py

## Run
python main.py
